IT 용어 50선

IT 분야에서 자주 사용되는 용어 50개 이상의 설명입니다.

IT 용어 목록

1. HTML
웹 페이지의 구조를 정의하는 마크업 언어입니다.
2. CSS
웹 페이지의 스타일과 레이아웃을 지정하는 언어입니다.
3. JavaScript
웹 페이지에 동적인 기능을 추가하는 프로그래밍 언어입니다.
4. 웹 브라우저
웹 페이지를 표시해주는 소프트웨어입니다.
5. 서버
클라이언트에게 데이터를 제공하는 컴퓨터 또는 소프트웨어입니다.
6. 클라이언트
서버에 요청을 보내고 응답을 받는 컴퓨터 또는 소프트웨어입니다.
7. URL
웹 자원의 위치를 나타내는 주소입니다.
8. HTTP
웹에서 데이터를 주고받기 위한 프로토콜입니다.
9. HTTPS
보안이 강화된 HTTP 프로토콜입니다.
10. 도메인
웹사이트의 주소를 나타내는 문자열입니다.
11. IP 주소
네트워크 상에서 컴퓨터를 식별하는 고유한 번호입니다.
12. DNS
도메인 이름을 IP 주소로 변환해주는 시스템입니다.
13. FTP
파일을 전송하기 위한 프로토콜입니다.
14. 데이터베이스
데이터를 체계적으로 저장하고 관리하는 시스템입니다.
15. SQL
데이터베이스를 관리하기 위한 언어입니다.
16. API
프로그램 간에 데이터를 주고받기 위한 인터페이스입니다.
17. 프론트엔드
사용자가 직접 보는 웹의 화면 부분을 개발하는 영역입니다.
18. 백엔드
서버, 데이터베이스 등 사용자에게 보이지 않는 부분을 개발하는 영역입니다.
19. 프레임워크
소프트웨어 개발을 쉽게 해주는 뼈대나 구조입니다.
20. 라이브러리
자주 사용하는 기능을 모아둔 코드 집합입니다.
21. 오픈소스
소스코드가 공개되어 누구나 사용할 수 있는 소프트웨어입니다.
22. 깃(Git)
소스코드의 버전을 관리하는 시스템입니다.
23. 깃허브(GitHub)
깃 저장소를 온라인에서 관리할 수 있는 서비스입니다.
24. IDE
통합 개발 환경으로, 개발에 필요한 도구를 모아놓은 소프트웨어입니다.
25. 컴파일러
프로그래밍 언어를 기계어로 번역해주는 프로그램입니다.
26. 인터프리터
프로그래밍 언어를 한 줄씩 해석하여 실행하는 프로그램입니다.
27. 알고리즘
문제를 해결하기 위한 절차나 방법입니다.
28. 자료구조
데이터를 효율적으로 저장하고 관리하는 방법입니다.
29. 객체지향
객체 단위로 프로그램을 구성하는 개발 방법입니다.
30. 함수형 프로그래밍
함수를 중심으로 프로그램을 구성하는 개발 방법입니다.
31. REST
웹에서 자원을 정의하고 자원에 대한 주소를 지정하는 아키텍처 스타일입니다.
32. 쿠키
웹 브라우저에 저장되는 작은 데이터 조각입니다.
33. 세션
사용자와 서버 간의 상태 정보를 유지하는 방법입니다.
34. 캐시
자주 사용하는 데이터를 임시로 저장해두는 공간입니다.
35. CDN
콘텐츠를 빠르게 전달하기 위한 전 세계 서버 네트워크입니다.
36. 클라우드
인터넷을 통해 IT 자원을 제공하는 서비스입니다.
37. 가상화
하드웨어 자원을 논리적으로 분할하여 사용하는 기술입니다.
38. 컨테이너
애플리케이션을 격리된 환경에서 실행하는 기술입니다.
39. 도커(Docker)
컨테이너 기반의 가상화 플랫폼입니다.
40. 머신러닝
컴퓨터가 데이터를 통해 스스로 학습하는 기술입니다.
41. 인공지능(AI)
컴퓨터가 인간처럼 사고하고 학습하는 기술입니다.
42. 빅데이터
대용량의 데이터를 분석하고 활용하는 기술입니다.
43. IoT
사물에 센서를 부착해 인터넷으로 연결하는 기술입니다.
44. 블록체인
데이터를 분산 저장하여 위변조를 막는 기술입니다.
45. 암호화
데이터를 안전하게 보호하기 위해 내용을 변환하는 기술입니다.
46. 해시
임의의 데이터를 고정된 크기의 값으로 변환하는 함수입니다.
47. SSL
인터넷에서 데이터를 암호화하여 안전하게 전송하는 프로토콜입니다.
48. 방화벽
네트워크를 보호하기 위해 외부 접근을 차단하는 시스템입니다.
49. 백업
데이터를 복사하여 안전하게 보관하는 작업입니다.
50. 복원
백업된 데이터를 다시 원래 상태로 되돌리는 작업입니다.
51. VPN
인터넷상에서 안전하게 통신할 수 있도록 해주는 가상 사설망입니다.
52. 와이파이(Wi-Fi)
무선으로 인터넷에 접속할 수 있는 기술입니다.
53. 블루투스
근거리 무선 통신 기술입니다.
54. USB
컴퓨터와 주변기기를 연결하는 표준 인터페이스입니다.
55. SSD
반도체를 이용한 저장장치로, HDD보다 빠릅니다.
56. HDD
자기 디스크를 이용한 전통적인 저장장치입니다.
57. RAM
컴퓨터가 작업 중에 사용하는 임시 저장장치입니다.
58. ROM
데이터가 영구적으로 저장되는 메모리입니다.
59. 운영체제(OS)
컴퓨터 하드웨어와 소프트웨어를 관리하는 시스템 소프트웨어입니다.
60. 드라이버
하드웨어를 제어하기 위한 소프트웨어입니다.
